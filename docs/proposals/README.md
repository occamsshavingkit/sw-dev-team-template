# docs/proposals/

Paired Markdown proposals for AI-loop changes to the protected-files set and to the customer-truth set, per FR-027 in the active spec. Classification: **ephemeral** — proposals live here while under deliberation and are removed (or archived per the project's retention rule) once the corresponding amendment lands in the canonical artifact. Each proposal cites the originating task ID, the target set, and the reviewing roles. This directory is not a long-term decision log; ADRs and customer-truth entries remain the durable records.
