# `docs/tech-lead/`

Reserved for `tech-lead`-authored orchestration content (working
notes, dispatch stubs, session ledgers) that doesn't fit
`docs/OPEN_QUESTIONS.md`, `docs/intake-log.md`, or `docs/DECISIONS.md`.

Allow-listed at `docs/tech-lead/**` by
`scripts/hooks/tech-lead-authoring-guard.py` (FW-ADR-0012).
