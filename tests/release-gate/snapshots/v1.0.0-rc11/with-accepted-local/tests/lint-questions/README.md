# tests/lint-questions/

Fixture corpus driving `scripts/lint-questions.sh`, which enforces the customer-question style rules. Classification: **canonical** — fixtures are the executable definition of "good" and "bad" question shapes. Layout: one good and one bad example per pattern, named `<pattern>-good.md` and `<pattern>-bad.md`. Reference: FR-012 (spec.md), R-8 (research.md).
